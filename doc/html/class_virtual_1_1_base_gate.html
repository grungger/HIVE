Detailed Description

BaseGate is the ABC base class of all gates.

The BaseGate class is an abstract class modeling all the methods used by the derived classes including Gate, MicroCircuit, byteMicroCircuit. All methods are set to virtual but the ones that only used by selected derived classes have a default implementation.
